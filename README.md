## projects

### [RedeemSoul](https://github.com/zaacbouy-afk/RedeemSoul)
> Windows hotkey productivity tool

One keypress closes all your open windows, opens a fresh browser with your chosen tabs, and launches your apps. Fully configurable — set your trigger key, which windows to keep, which URLs to open, and which apps to launch. Windows only.

`Python` `pygetwindow` `keyboard` `appopener`

### [Apollo-Bot](https://github.com/zaacbouy-afk/Apollo-Bot)
> Open-source Discord bot

Built with Python and py-cord. Includes YouTube music playback with queuing, slash commands, message purging, basic auto-moderation (bad words, warnings, timeouts), and hot-reloadable cogs. Public so others can learn from it and run their own copy.

`Python` `py-cord` `yt-dlp` `FFmpeg`
